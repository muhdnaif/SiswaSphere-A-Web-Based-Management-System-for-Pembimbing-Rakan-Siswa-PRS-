# SiswaSphere

**SiswaSphere** is a web-based management system designed for **Pembimbing Rakan Siswa (PRS)**, a university student leadership club. Built with **Laravel** and **MySQL**, this system centralizes administrative tasks, improving efficiency, organization, and collaboration among club members.

## Features

- **Member & Committee Management**: Easily manage PRS members, roles, and committee assignments.
- **Meeting Management**: Schedule meetings, track attendance, and document meeting notes.
- **Secure Document Storage**: Upload and manage financial reports, official letters, and other important documents securely.
- **Centralized Data Management**: Streamline administrative workflows and improve organizational efficiency.

## Technologies Used

- **Backend**: Laravel (PHP Framework)
- **Database**: MySQL
- **Frontend**: Blade templating engine (Laravel) + HTML/CSS
- **Authentication & Security**: Built-in Laravel Auth system
